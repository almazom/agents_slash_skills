# Примеры

Считай эти формулировки каноническими примерами для `kiselev-read`.

Важно:

- `kiselev-week` это shortcut внутри `kiselev-read`
- он всегда означает недельный read-only отчёт по DEKSDEN в сохранённом формате
- этот shortcut никогда не относится к ИИшнице

## Забрать Оба Источника

Важно:

Внутри `kiselev-read` слова `оба источника` всегда означают:

- `@deksden_notes`
- `DEKSDEN (chat)`

и никогда не означают `ИИшница`.

Пользователь:

`kiselev-read: забери сегодня и чат и канал`

Смысл:

- забрать и `@deksden_notes`, и `DEKSDEN (chat)`
- остаться в read-only режиме
- подготовить общую выборку для следующего анализа

Пользователь:

`kiselev-read: забери канал и чат за неделю`

Смысл:

- забрать 7-дневное окно для обоих источников
- разложить снапшоты для таймлайна и аналитики

Пользователь:

`kiselev-read: перезабери оба источника и потом проанализируй`

Смысл:

- сначала обновить снапшоты по двум источникам
- потом перейти к общему dedupe-анализу

Пользователь:

`kiselev-read: сначала забери потом проанализируй оба источника за неделю простым русским`

Смысл:

- `оба источника` здесь означают именно DEKSDEN канал и DEKSDEN чат
- сначала сделать недельный fetch по двум DEKSDEN источникам
- потом сделать общий недельный анализ простым русским

## Показать Сообщения Из Обоих Источников

Пользователь:

`kiselev-read: покажи последние сообщения из чата и канала`

Смысл:

- взять свежие данные из обоих источников
- показать короткий split-view:
  последние посты канала
  последние сообщения чата

Пользователь:

`kiselev-read: покажи последний пост канала и последние сообщения чата`

Смысл:

- показать самую новую публикацию канала
- показать свежие реальные сообщения чата вокруг неё

## Последнее Сообщение

Пользователь:

`kiselev-read: объясни последнее сообщение простым русским`

Смысл:

- взять свежие данные
- понять, последнее это сообщение из канала или из чата
- объяснить с контекстом

## Один Человек

Пользователь:

`kiselev-read: объясни последнее сообщение от @michaelsavin`

Смысл:

- искать в общем read-only пространстве
- если есть общий dedupe-набор, опираться на него
- объяснить простым русским

## Аналитика За День

Пользователь:

`kiselev-read: аналитика за 1 день`

Смысл:

- взять или обновить сегодняшние снапшоты
- собрать общий dedupe-набор
- вернуть короткий аналитический отчёт

Пользователь:

`kiselev-read: проанализируй сегодня и чат и канал`

Смысл:

- рассматривать оба источника как одно пространство коммуникации
- сохранять пометки, откуда что пришло
- отделять оригинальные посты канала от реакции в чате

Пользователь:

`kiselev-read: сначала забери потом проанализируй оба источника за сегодня простым русским`

Смысл:

- сначала сделать fetch
- потом запустить общий dedupe-анализ
- ответить простым русским

## Отчёт За Неделю

Пользователь:

`kiselev-week: сделай недельный отчёт`

Смысл:

- это shortcut на фиксированный недельный шаблон
- сначала взять недельный combined dataset
- потом выдать ответ в сохранённой структуре:
  кратко
  что публиковали в канале
  обсуждения в чате
  цитаты недели

Пользователь:

`kiselev-week: сначала забери неделю и собери отчёт`

Смысл:

- сначала освежить данные по двум источникам
- потом собрать недельный отчёт именно в сохранённом формате `kiselev-week`

Пользователь:

`kiselev-read: сделай недельный отчёт по чату и каналу вместе`

Смысл:

- собрать общий отчёт за 7 дней
- показать поток публикаций из канала
- показать поток реакции и обсуждения из чата

Пользователь:

`kiselev-read: проанализируй оба источника за последние 7 дней`

Смысл:

- это тот же недельный общий отчёт
- зеркальные посты по умолчанию убирать через dedupe

Пользователь:

`kiselev-read: сначала забери неделю потом сделай отчёт по двум источникам`

Смысл:

- сначала сделать недельный fetch
- потом собрать общий недельный отчёт
- считать `канал + чат` режимом по умолчанию

## Недельный Fetch

Пользователь:

`kiselev-read: забери неделю по каналу и чату`

Смысл:

- забрать 7 дней сразу по двум источникам
- подготовить weekly snapshots для следующего анализа

Пользователь:

`kiselev-read: обнови неделю по двум источникам`

Смысл:

- освежить недельные данные
- потом использовать их как основу для отчёта, таймлайна и extract-задач

## Недельные Сообщения

Пользователь:

`kiselev-read: покажи главные сообщения за неделю из канала и чата`

Смысл:

- взять weekly combined dataset
- показать самые важные публикации канала и самые заметные реакции чата

Пользователь:

`kiselev-read: покажи неделю по двум источникам коротко`

Смысл:

- сделать сжатый weekly overview
- без длинного отчёта
- с сохранением разделения на канал и чат

## Таймлайн За Неделю

Пользователь:

`kiselev-read: хочу таймлайн за 1 неделю`

Смысл:

- работать по общему dedupe-набору за 7 дней
- делать акцент на хронологии
- показывать моменты публикаций в канале и волны реакции в чате

Пользователь:

`kiselev-read: сделай недельный таймлайн по каналу и чату`

Смысл:

- это тот же weekly timeline
- нужно явно показать, что пришло из канала и что было реакцией чата

## Ссылки За Неделю

Пользователь:

`kiselev-read: собери ссылки за неделю из чата и канала`

Смысл:

- использовать недельный общий набор
- собрать ссылки из обоих источников
- показать, какие ссылки были в постах канала, а какие принёс чат

Пользователь:

`kiselev-read: дай недельную сводку по ссылкам из двух источников`

Смысл:

- тот же weekly links report
- сделать акцент на внешних ресурсах за неделю

## Ссылки

Пользователь:

`kiselev-read: собери ссылки за сегодня`

Смысл:

- использовать дневной набор
- вытащить и дедуплицировать URL
- указать, что пришло из канала, а что из чата

## GitHub

Пользователь:

`kiselev-read: проанализируй все github ссылки за неделю`

Смысл:

- отфильтровать GitHub URL из общего недельного набора
- сохранить контекст вокруг каждой ссылки

Пользователь:

`kiselev-read: github за неделю по чату и каналу вместе`

Смысл:

- собрать GitHub ссылки из двух источников
- проанализировать их по общему недельному набору

## Темы За Неделю

Пользователь:

`kiselev-read: выдели темы недели по двум источникам`

Смысл:

- взять недельный общий набор
- выделить главные темы
- показать, где тема родилась: в канале или в чате

Пользователь:

`kiselev-read: за неделю сфокусируйся на теме agents`

Смысл:

- сделать недельный тематический срез
- искать тему сразу в канале и в чате

## Один Человек За Неделю

Пользователь:

`kiselev-read: за неделю найди сообщения от @michaelsavin и объясни`

Смысл:

- искать человека в weekly combined dataset
- брать контекст вокруг его сообщений
- объяснять простым русским

## Только Один Источник

Пользователь:

`kiselev-read: только канал за сегодня`

Смысл:

- работать только с `deksden_notes`

Пользователь:

`kiselev-read: только чат за сегодня`

Смысл:

- работать только с `kiselev_chat`

## Важный Guardrail

Если формулировка случайно звучит как:

- отправь
- нажми
- опубликуй
- ответь в Telegram

не делать это внутри этого навыка.
